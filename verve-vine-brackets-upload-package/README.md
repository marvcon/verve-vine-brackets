# Verve & Vine March Madness Site

## Quick start
1. Unzip this project.
2. Open a terminal in the project folder.
3. Run:
   npm install
4. Start locally:
   npm run dev
5. Open:
   http://localhost:3000

## Deploy to Vercel
1. Create a free GitHub repo.
2. Upload all project files.
3. In Vercel, import the GitHub repo.
4. Click Deploy.

## Before going live
Update these parts in `pages/index.js`:
- Replace the premium CTA button with your Gumroad or checkout link.
- Connect the free download form to your email provider.
- Add your real file delivery links.

## Project structure
- `pages/index.js` - main site
- `pages/_app.js` - global app wrapper
- `styles/globals.css` - styles
- `tailwind.config.js` - Tailwind config
- `postcss.config.js` - PostCSS config
