## Run locally
1. Unzip the project
2. Open terminal in the project folder
3. Run:
   npm install
4. Then:
   npm run dev

## Redeploy
1. Replace your repo files with these
2. Commit and push
3. Vercel should auto-deploy
4. If needed: Vercel -> Deployments -> Redeploy
